Beuty-Salon 💇‍♀️💅
Projeto de gerenciamento de salão de beleza desenvolvido como trabalho prático para a disciplina de Banco de Dados 2, sob a orientação do Professor Eduardo Chagas.

🎓 Contexto Acadêmico
Instituição: Uniube - Universidade de Uberaba

Curso: Sistemas de Informação

Disciplina: Banco de Dados 2

Professor: Eduardo Chagas

Propósito: Aplicação de conceitos avançados de modelagem, integridade de dados e persistência em um cenário de software real.

🚀 Funcionalidades
O sistema foi projetado para atender às necessidades de um salão de beleza, focando na organização dos dados:

Gestão de Clientes: Cadastro e histórico de visitas.

Catálogo de Serviços: Controle de procedimentos, preços e duração.

Agendamentos: Vínculo entre clientes, serviços e profissionais.

Persistência de Dados: Toda a lógica de negócio é suportada por um banco de dados relacional.

🛠️ Tecnologias Utilizadas
Linguagem: C# / .NET

Ambiente de Desenvolvimento: Visual Studio

Banco de Dados: SQL Server (ou banco utilizado no curso)

Versionamento: Git & GitHub

🗄️ Foco em Banco de Dados (SI)
Como requisito da disciplina de Banco de Dados 2 no curso de Sistemas de Informação, o projeto destaca:

Modelagem Relacional: Estrutura normalizada para garantir a integridade dos dados.

Integridade Referencial: Uso rigoroso de Chaves Primárias (PK) e Chaves Estrangeiras (FK).

Consultas e Scripts: Implementação de buscas otimizadas para o gerenciamento de horários e serviços.

⚙️ Como executar o projeto
Clone este repositório:

Bash

git clone https://github.com/AlfredoVentura/Beuty-Salon.git
Abra no Visual Studio: Localize o arquivo .sln ou .slnx na raiz da pasta.

Configure o Banco de Dados: * Verifique a ConnectionString no arquivo de configuração do projeto.

Certifique-se de que o SQL Server está rodando localmente.

Inicie a aplicação: Pressione F5.
